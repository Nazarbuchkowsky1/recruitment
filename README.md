# Telegram-бот для працевлаштування

Бот для пошуку роботи за кордоном з повним функціоналом згідно з технічним завданням.

## 🚀 Запуск бота

1. Встановіть залежності:
```bash
pip install -r requirements.txt
```

2. Отримайте свій Chat ID:
   - Напишіть боту @userinfobot
   - Скопіюйте ваш Chat ID

3. Відредагуйте файл `main.py`:
   - Знайдіть рядок `ADMIN_CHAT_ID = "YOUR_ADMIN_CHAT_ID"`
   - Замініть `YOUR_ADMIN_CHAT_ID` на ваш Chat ID

4. Запустіть бота:
```bash
python main.py
```

## 📋 Функціональність

### 🧾 Вакансії
- 5 готових вакансій з детальним описом
- Кожна вакансія містить: опис, вимоги, умови, контакти
- Кнопка "Подати заявку" для кожної вакансії

### ✅ Гарантії / Відгуки
- 20 відгуків працівників
- Навігація кнопками "Попередній" / "Наступний"
- Автоматичне зациклювання

### 📍 Контакти
- Повна контактна інформація
- Телефони, email, Telegram менеджери
- Адреса офісу та посилання на карту
- Час роботи

### 📄 Документи
- Інформація про ліцензії та сертифікати
- Список документів що підтверджують діяльність
- Посилання на відео та додаткові матеріали

### 📨 Заявка на працевлаштування
Повна анкета з 5 кроками:
1. Ім'я та прізвище
2. Номер телефону
3. Місто
4. Вибір вакансії
5. Досвід роботи та коментарі

Заявки автоматично надсилаються адміну.

### ℹ️ Як ми працюємо
Детальний опис етапів працевлаштування:
- Подача заявки
- Дзвінок менеджера
- Підготовка документів
- Співбесіда
- Підписання договорів
- Виїзд та початок роботи

## 🛠️ Налаштування

### Зміна вакансій
Відредагуйте масив `VACANCIES` в файлі `main.py`

### Зміна відгуків
Відредагуйте масив `REVIEWS` в файлі `main.py`

### Зміна контактів
Знайдіть функцію `button_handler` і відредагуйте розділ `contacts`

### Зміна документів
Знайдіть функцію `button_handler` і відредагуйте розділ `documents`

## 📱 Використання

1. Користувач натискає `/start`
2. Обирає потрібний розділ з головного меню
3. Може переглядати вакансії, відгуки, контакти
4. Може подати заявку через анкету
5. Заявки надсилаються адміну автоматично
